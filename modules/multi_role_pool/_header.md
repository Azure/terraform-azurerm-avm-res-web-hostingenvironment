# Multi-Role Pool (Front-End Pool)

This submodule manages the multi-role pool (front-end pool) for an App Service Environment using the AzAPI provider.

> **Note:** This resource is primarily used for ASEv1/v2. ASEv3 manages front-end pools internally and this resource may not be applicable.

## Features

- Configures the front-end pool compute mode
- Sets worker count and size
- Supports SKU configuration for scaling
